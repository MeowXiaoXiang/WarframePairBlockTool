# WarframePairBlockTool
 Warframe配對阻止工具，可限制Warframe 特定UDP輸出 來達到成為主機的效果
## 打包所需說明<br/>
* 安裝 pyinstaller<br/>
    * pip install pyinstaller
* 將main.py打包,在終端輸入下列指令(要在同個目錄下)
  * pyinstaller -F -w main.py
* 打包好後的檔案將會放在多出來的dist資料夾中即可使用
